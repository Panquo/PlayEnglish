In this section you'll learn about us but not only !
PlayEnglish is a fictious website created for an english project. This project was initiated by english teachers of the Engineering school of TelecomNancy in France. The goal was to create an IT tool to help people learn english. In our case, we decided to create a website including various types of fun games to learn and revise english vocabulary.

We decided to develop these games because it was fun for us to code them, but it is also supposed to be fun to play. And because it is better to learn if you're interesting in the subject and you enjoy it, it was abvious that we would go for online games.

We are two developers to code on this website : Merlin Baudou and Frantz Darbon. We are both students at TelecomNancy and learning and practicing new languages. But for this project we had barely touched to all the languages that can include a website, so we had to learn them specificaly for this project. Merlin was in charge of the major part of HTML and CSS, which is how the website looks like, and Frantz was designing the games in Javascript, but we both touched to all the different languages.

If you want to learn more about us, you can send us a message via the contact page, we would be pleased to answer any of your questions.
